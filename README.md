# IT Asset Manager

## Overview

IT Asset Manager is a web application designed to help manage company IT equipment and user accounts. It allows administrators to keep track of hardware assets, assign equipment to users, manage user roles, and monitor asset availability.

The application is built using HTML, CSS, and JavaScript and stores all data locally in the browser using Local Storage.

---

## Features

* Dashboard with asset statistics
* Add new IT assets
* View all company equipment
* Search assets
* Manage asset status
* Assign assets to users
* Add new users
* Search users
* Manage user roles
* Promote users to different IT roles
* View assigned equipment for each user
* Delete users
* Delete assets
* Automatic statistics updates
* Local Storage support (data persists after refreshing the page)
* Responsive interface

---

## Asset Information

Each asset contains:

* Asset name
* Serial number
* Asset type
* Status
* Location
* Assigned user

Supported asset types include:

* Laptop
* Monitor
* Phone
* Printer
* Server
* Network Device

Supported asset statuses:

* Available
* Assigned
* Repair
* Retired

---

## User Management

Each user contains:

* Full name
* Email address
* Department
* Role

Available roles include:

* Employee
* IT Support
* Linux Admin
* Windows Admin
* Network Admin
* Security Admin

Administrators can change user roles at any time.
